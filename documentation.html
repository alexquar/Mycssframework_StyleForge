Using Colours

Text Colour

To use one of our colours on the font please use: text-colourname

Text Hover

To make your text change colours on mouse hover over please use: text-hover-colourname

Background Colours

To make change the colour of an elements background please use: bg-colourname.

Border Colours

To change the colour of an elements border please use: border-colourname.

Utility Classes

Padding

To place padding on an item use the following:

p/pl/pr/pb/pt - 1/2/3/4/5

all/left/right/bottom/top

Margin

To place padding on an item use the following:

m/ml/mr/mb/mt - 1/2/3/4/5

all/left/right/bottom/top

Opacity

To change the opacity of an element please use the following:

o-10/----/100

Border Radius

To place padding on an item use the following:

br-default/none/xs/sm/lg/full

Border Width

To modify the border width of an item use the following:

bw - 1/2/3/4/5/10/25

1px --- 25px

Border Style

To modify the border style of an item use the following:

bs - s/da/do/g/r

solid/dashed/dotted/groove/ridge

Font Size

To change the size of text use the following:

font-size - sm/md/lg/xl

Display

To apply different display properties use the following:

display-n/f/b/i/i-o

none/flex/block/inline/inline-block

Grid

Getting started

To get started create a div with a class of container

Then create divs with the class of row to define the rows you want your grid split into

Each row will stretch a standard 12 columns.

Columns

Inside your rows define your content inside divs with classes set up like this to determine how much space they take up.

class = "col-n-breakpoint

n can be any number less than 12. If n is 12 the div will take up 100% percent of the row. If n is 6 the div will take up 50% of the row. And so on ...

The breakpoint is the screen width at which you want this size of the div to take effect. Breakpoint options are xs,sm,md,lg,xl. You can define a size for the dive at all the breakpoints

Take a look at the example below by changing the size of your screen.

Columns

This will take up 3 cols on xs screens, 4 on sm, 6 on md, 8 on lg and 12 on xl!

Column Gap

To add a gap inbetween the divs of your rows use this on your row div:

class= "row gap-n"

n can be between 1 and 5.

Justifying Space

To determine where the empty space in your rows is place use this on your row div:

class = "row justify-options

option can be flex-start/flex-end/space-between/space-around.

Cards

Working with Cards

Cards are what this documentation page is built from!

To create a card place you content inside a div with a class of card

Inside there create your card title with a class of card-title and any body text with a class of card-body

Then, customize!!!

Buttons

Basic Button

To create a basic button and style it add a class of the following form:.

class = "btn-colourname"

Button Outline

To create an outlined button add a class of the following form:

class = "btn-outline-colourname"

Compliment Buttons

To create a funky looking compliment button add a class of the following form:

class = "btn-compliment-colourname"
